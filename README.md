# 極簡加密資產儀表板

一款**極簡、易用、隱私優先**的加密貨幣資產整合平台。在一個地方安全地管理您在 Binance、OKX 等多個交易所的資產。

## ✨ 核心特性

### 🔒 安全第一

- **AES-256 加密**：您的 API Key 使用軍用級加密儲存
- **唯讀 API**：我們無法進行交易、提款或任何資金操作
- **零知識設計**：我們無法訪問您的私鑰或助記詞

### 📊 資產整合

- **多交易所支援**：Binance、OKX（更多交易所持續新增）
- **多鏈錢包追蹤**：Ethereum、BSC、Polygon、Arbitrum、TRON
- **實時幣價**：自動從 CoinGecko 獲取最新價格
- **雙幣種顯示**：USD 和 TWD 雙幣種資產總值

### ⚡ 自動化同步

- **定期更新**：每 10 分鐘自動同步一次資產數據
- **快照歷史**：保留 90 天的資產快照，追蹤資產變化
- **錯誤提示**：同步失敗時立即通知，方便排查問題

### 🌐 華語優先

- **繁體中文介面**：完全本地化的用戶體驗
- **易用設計**：簡潔直觀，無需複雜配置

## 🚀 快速開始

### 系統要求

- 現代瀏覽器（Chrome、Firefox、Safari、Edge）
- 穩定的網路連接
- 交易所的 API Key（需要「唯讀」權限）

### 第一步：登入

1. 訪問應用首頁
2. 點擊「登入開始」按鈕
3. 使用 Manus 帳號授權

### 第二步：連接交易所

1. 進入「交易所 API 管理」頁面
2. 點擊「新增 API Key」
3. 選擇交易所（Binance 或 OKX）
4. 輸入您的 API Key 和 Secret
5. 點擊「連接」

**如何獲取 API Key？**

#### Binance

1. 登入 [Binance 官網](https://www.binance.com)
2. 進入「帳號」→「API 管理」
3. 建立新的 API Key
4. 編輯限制：
   - ✅ 啟用「讀取」權限
   - ❌ 禁用「交易」和「提現」權限
5. 複製 API Key 和 Secret

#### OKX

1. 登入 [OKX 官網](https://www.okx.com)
2. 進入「帳號」→「API」
3. 建立新的 API Key
4. 設定權限：
   - ✅ 只選擇「讀取」權限
   - ❌ 不要選擇「交易」和「提現」
5. 複製 API Key、Secret 和 Passphrase

### 第三步：追蹤錢包（可選）

1. 進入「錢包地址管理」頁面
2. 點擊「新增錢包」
3. 選擇區塊鏈網路
4. 輸入您的公開錢包地址
5. 點擊「新增」

**支援的區塊鏈：**
- Ethereum（以太坊）
- BSC（幣安智能鏈）
- Polygon
- Arbitrum
- TRON

### 第四步：查看資產

1. 返回儀表板主頁
2. 查看「總資產價值」卡片
3. 查看「資產持倉」列表
4. 點擊「立即同步」手動更新數據

## ❓ 常見問題

### Q: 您能看到我的 API Key 嗎？

**A:** 不能。您的 API Key 使用 AES-256 加密儲存，我們無法解密查看。

### Q: 您能進行交易嗎？

**A:** 不能。我們只使用「唯讀」API 權限，技術上無法進行任何交易、提款或轉帳。

### Q: 如果我的 API Key 被洩露怎麼辦？

**A:** 立即在交易所後台重新生成 API Key，然後在本應用中刪除舊的 Key 並新增新的。

### Q: 我的數據會保留多久？

**A:** 資產快照保留 90 天，訪問日誌保留 30 天。刪除帳號後，所有數據立即清除。

### Q: 為什麼同步失敗了？

**A:** 可能的原因：
- API Key 無效或已過期
- 交易所 API 被速率限制
- 網路連接問題

查看「最後同步錯誤」信息了解詳細原因。

### Q: 支援哪些交易所？

**A:** 目前支援：
- ✅ Binance（幣安）
- ✅ OKX

更多交易所正在開發中。

### Q: 支援哪些錢包？

**A:** 目前支援以下區塊鏈上的公開地址：
- ✅ Ethereum
- ✅ BSC
- ✅ Polygon
- ✅ Arbitrum
- ✅ TRON

### Q: 如何更新我的帳號信息？

**A:** 目前無法直接更新。如需修改，請刪除帳號後重新建立。

### Q: 應用支援手機嗎？

**A:** 是的。應用採用響應式設計，完全支援手機、平板和桌面。

## 🔐 安全建議

1. **定期檢查 API Key**
   - 每月檢查一次交易所的 API Key 設定
   - 確認沒有陌生的 IP 訪問過

2. **使用強密碼**
   - 為您的交易所帳號設定強密碼
   - 不要使用簡單或常見的密碼

3. **啟用雙因素認證**
   - 在交易所啟用 2FA（如果支援）
   - 使用認證器應用而不是短信

4. **不要分享 API Key**
   - 不要在公開場合分享 API Key
   - 不要將 API Key 保存在不安全的地方

5. **定期輪換 API Key**
   - 每 3 個月重新生成一次 API Key
   - 刪除舊的 Key 並新增新的
