# 極簡加密資產儀表板 - 開發待辦清單

## 專案目標
打造一款極簡、易用、專為華語用戶設計，並將用戶數據隱私做到極致的加密貨幣資產整合儀表板。

## 核心設計原則
- **安全性優先**：API Key 加密儲存，強調唯讀權限
- **隱私優先**：用戶數據完全加密，絕不與第三方分享
- **極簡易用**：3 分鐘內完成註冊和資產同步
- **自動化同步**：後端每 5-10 分鐘自動更新資產數據
